Automatic ID Generation

Wealthy Life lets you automatically assign Patient IDs for newly added patients to your records. You can also assign Patient IDs to all your existing patients.

Current Pattern: P {{PATIENT-NUMBER}} edit

You need to select an Initial Value. You can choose an alphanumeric prefix (will not be increment), and a numeric part which will be automatically incremented for every new bill.

Current Pattern: INV {{INVOICE-NUMBER}} edit

You need to select an Initial Value. You can choose an alphanumeric prefix (will not be increment), and a numeric part which will be automatically incremented for every new receipt.
